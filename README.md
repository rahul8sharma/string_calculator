# String Calculator

This is a simple implementation of a **String Calculator**. It sums a string of numbers, which can be separated by commas, newlines, or custom delimiters. It also checks for negative numbers and throws an exception if any are found.

## Requirements

- Ruby (>= 2.6)
- Bundler
- RSpec (for testing)

## Installation

1. **Clone the repository**:

   ```bash
   git clone https://github.com/your-username/string_calculator.git
   cd string_calculator
   
1. **run the rspec**:

   ```bash
   bundle 
   bundle exec rspec spec/string_calculator_spec.rb
